# Scheduling Task

Create:

```bash
spring init --artifact-id "scheduling-task" --group-id "org.jfcmc" --version "0.0.1" --description "Scheduled Tasks" --package-name "org.jfcmc.springboot.scheta" --name "Sheduled Tasks" --boot-version 4.0.3 --build maven --language java --java-version 17 --dependencies scheduling-task
```

Compile:

```bash
mvn compile
```

Executing:

```bash
mvn spring-boot:run
```
