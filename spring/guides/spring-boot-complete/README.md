# Application with Spring Boot

Creating an application

```bash
spring init --artifact-id "spring-boot-complete" --group-id "org.jfcmc" --version "0.0.1" --description "Application with Spring Boot" --package-name "org.jfcmc.spring.springbootcomplete" --name "spring-boot-complete" --boot-version 4.0.3 --build maven --language java --java-version 17 --dependencies "web,actuator"  spring-boot-complete/
```
